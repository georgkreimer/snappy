# Snappy

Takes a snapshot with your Mac's built-in iSight/FaceTime webcam by command line.

## Install

    git clone 
    cd snappy/
    rake compile
    rake install
    
## Usage

    snappy file.jpg

See `snappy -h` to get more help.
